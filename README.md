# 数学练习题生成器

一个简单的网页应用，用于生成小学数学练习题并支持打印。

## 功能

- 支持加法、减法、乘法、除法及混合运算
- 题目类型：
  - 加减混合
  - 纯加法
  - 纯减法
  - 纯乘法
  - 纯除法
  - 乘除混合
  - 四则混合
- 加减法和乘除法分别设置数值范围
  - 加减法：操作数在设定范围内
  - 乘除法：乘数/被乘数、除数/商在设定范围内，确保整除
- 屏蔽数字功能：可排除特定数字（如 1,10）避免出现过于简单的题目
- 多种答案样式：方框、下划线、括号
- 可自定义题目数量和列数
- 一键显示/隐藏答案
- 优化打印布局，适合A4纸张

## 使用方法

1. 直接用浏览器打开 `index.html`
2. 设置题目类型、数字范围、题目数量等参数
3. 可选：在"屏蔽数字"中输入要排除的数字（逗号分隔）
4. 点击"生成新题目"按钮
5. 使用浏览器打印功能(Ctrl+P)打印

## 文件结构

- `index.html` - 页面结构
- `script.js` - 题目生成逻辑
- `style.css` - 样式和打印布局
