# 数学练习题生成器

一个简单的网页应用，用于生成小学数学练习题并支持打印。

## 功能

- 支持加法、减法、乘法、除法及混合运算
- 可自定义数字范围、题目数量和列数
- 除法题目为2位数除以1位数，确保整除
- 多种答案样式：方框、下划线、括号
- 一键显示/隐藏答案
- 优化打印布局，适合A4纸张

## 使用方法

1. 直接用浏览器打开 `index.html`
2. 设置题目类型、数字范围、题目数量等参数
3. 点击"生成题目"按钮
4. 使用浏览器打印功能(Ctrl+P)打印

## 文件结构

- `index.html` - 页面结构
- `script.js` - 题目生成逻辑
- `style.css` - 样式和打印布局
